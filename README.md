# Laboratorio 03: Diseño de Casos de Prueba Funcionales

Este repositorio contiene el desarrollo completo del Laboratorio 03 para la asignatura de **Pruebas y Aseguramiento de Calidad de Software (IS-489)** de la Universidad Nacional de San Cristóbal de Huamanga.

## Contenido del Repositorio

De acuerdo con las indicaciones de la cátedra, el proyecto se ha dividido en dos secciones independientes:

### [1. Ejercicio de Laboratorio (SGI InkaRetail)](./laboratorio-sgi/)
Contiene el análisis de clases de equivalencia y la matriz de pruebas completada para el Sistema de Gestión de Inventario (SGI) de InkaRetail S.A.C. visto en clase.

### [2. Tarea para Casa (UNSCH Intranet)](./tarea-intranet/)
Contiene el diseño de los 10 casos de prueba funcionales aplicados al portal de la Intranet de la UNSCH, estructurado estrictamente bajo los 5 puntos solicitados en la guía.
