# Laboratorio 03: Diseño de Casos de Prueba Funcionales y Trazabilidad

Este repositorio contiene el desarrollo completo y la documentación técnica del **Laboratorio 03** para la asignatura de **Pruebas y Aseguramiento de Calidad de Software (IS-489)** de la Universidad Nacional de San Cristóbal de Huamanga.

## Contenido del Repositorio

De acuerdo con las indicaciones de la guía y aplicando estándares profesionales de QA, el proyecto se ha dividido en dos secciones independientes:

### [1. Ejercicio de Laboratorio: SGI InkaRetail](./laboratorio-sgi/)
Contiene el análisis de clases de equivalencia, edge cases y la matriz de pruebas completada para los módulos de Autenticación y Registro del Sistema de Gestión de Inventario (SGI) de InkaRetail S.A.C. visto en clase.

### [2. Proyecto Principal: UNSCH Intranet (Versión Mejorada)](./tarea-intranet/)
Contiene el diseño de pruebas funcionales aplicadas al portal de la Intranet de la UNSCH. **Esta entrega fue escalada más allá de los requisitos básicos, incorporando:**
* Cobertura sobre un ecosistema de **5 módulos críticos** (Matrícula, Notas, Pagos, Historial y Documentos).
* Ejecución de **12 Casos de Prueba** aplicando técnicas de caja negra (PE, AVL y Edge Cases).
* Construcción de una **Matriz de Trazabilidad de Requisitos (RTM)** cruzando Historias de Usuario (HUs) con Casos de Prueba (TCs).
* Detección y resolución técnica de **GAPs de cobertura**.
* Gestión y trazabilidad del proyecto replicada íntegramente en **Jira Software** (Epic ➔ Stories ➔ Sub-tasks).
